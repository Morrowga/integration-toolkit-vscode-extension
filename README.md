# ITK – Integration Toolkit

A VS Code extension for project integration & intelligence.

## Features

- **Dashboard Panel** – Real-time flow visualization of your project structure
- **Terminal Commands** – Built-in commands for scaffolding and deployment
- **Code Analysis** – Dependency mapping and issue detection

## Usage

1. Click the **ITK** icon in the Activity Bar
2. Or run `ITK: Open` from the Command Palette (`Cmd+Shift+P`)

## Commands

| Command | Description |
|---|---|
| `ITK: Open` | Open the ITK dashboard |
| `ITK: Clear Cache` | Clear cached project data |

## Requirements

- VS Code `^1.100.0`
